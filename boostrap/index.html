Utilice los estilos de botones personalizados de Bootstrap para acciones en formularios, cuadros de diálogo y más con soporte para múltiples tamaños, estados y más.


button tags

El .btn las clases están diseñadas para usarse con elemento. Sin embargo, también puedes utilizar estas clases en o elementos (aunque algunos navegadores pueden aplicar una representación ligeramente diferente). Cuando se utilizan clases de botones activadas los elementos que se utilizan para activar la funcionalidad en la página (como el contenido colapsado), en lugar de vincular a páginas nuevas o secciones dentro de la página actual, a estos enlaces se les debe dar un rol="botón" transmitir adecuadamente su propósito a tecnologías de asistencia como los lectores de pantalla.

outline buttons

¿necesitas un botón, pero no los fuertes colores de fondo que traen? Reemplace las clases de modificador predeterminadas con .btn-outline-* unos para eliminar todas las imágenes de fondo y colores de cualquier botón.


Disabled state

Haga que los botones parezcan inactivos agregando desactivado atributo booleano a cualquiera elemento. Los botones deshabilitados tienen eventos-puntero: ninguno aplicado para evitar que se activen los estados activos y flotantes.

De manera similar, cree menús desplegables de botones divididos con prácticamente el mismo marcado que los menús desplegables de un solo botón, pero con la adición de .dropdown-toggle-split para un espacio adecuado alrededor del control desplegable. Usamos esta clase extra para reducir la horizontalidad acolchado a cada lado del caret en un 25% y retire el margen izquierdo eso se agrega para los botones desplegables regulares. Esos cambios adicionales mantienen el intercalador centrado en el botón dividido y proporcionan un área de acierto de tamaño más apropiado al lado del botón principal.
